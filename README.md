# KelasC_Kelompok6_Perpustakaan

Sistem Manajemen Koleksi Perpustakaan

Program ini merupakan implementasi konsep **Object-Oriented Programming (OOP)** menggunakan bahasa pemrograman Python untuk mengelola data koleksi perpustakaan.

Sistem dapat mengelola beberapa jenis koleksi, yaitu:

- Buku
- Majalah
- Jurnal
- DVD Film Dokumenter

Program dikembangkan sebagai tugas mata kuliah **Pemrograman Berorientasi Objek (PBO)** dengan menerapkan konsep **Abstraction, Inheritance, dan Polymorphism**.

# Tujuan Program

Program ini dibuat untuk:

- Mengelola data koleksi perpustakaan secara terstruktur.
- Mempermudah proses penambahan, penghapusan, dan penampilan data koleksi.
- Mengimplementasikan konsep Object-Oriented Programming (OOP).
- Memahami penerapan inheritance, abstraction, dan polymorphism dalam studi kasus nyata.

------------------------------------------------

# Fitur Program

- Menambahkan data koleksi.
- Menghapus data koleksi berdasarkan kode koleksi.
- Menampilkan seluruh data koleksi.
- Mendukung berbagai jenis koleksi.
- Mudah dikembangkan untuk menambahkan jenis koleksi baru.
